# Exercícios de POO - Senac

Este repositório reúne exercícios simples de **Programação Orientada a Objetos (POO)** que eu desenvolvi durante meus estudos no **Senac**, com foco em praticar os conceitos fundamentais da orientação a objetos.

## Objetivo

O objetivo destes exercícios foi treinar, na prática:

- criação de classes e objetos
- atributos e métodos
- encapsulamento
- organização de código com responsabilidades separadas
- modelagem de problemas do dia a dia com POO

## Exercícios do repositório

- `01SistemaBancario.py` - Simulação de operações básicas de conta bancária.
- `02ControleDeProdutos.py` - Controle simples de produtos e informações relacionadas.
- `03SistemaDeFncionarios.py` - Modelagem de funcionários e suas características.
- `04SistemaDeVeiculos.py` - Estrutura de classes para veículos.
- `05Zoologico.py` - Representação de animais e comportamentos em um zoológico.
- `06FormasGeometricas.py` - Cálculos e modelagem de formas geométricas.
- `07SistemaDePagamento.py` - Simulação de regras básicas de pagamento.
- `08SistemaDeTransporte.py` - Modelagem de um sistema simples de transporte.

## Contexto

Esses projetos foram feitos com propósito educacional para reforçar minha base em POO, praticando raciocínio lógico e estruturação de código em Python.
