# OOP Exercises - Senac

This repository contains simple **Object-Oriented Programming (OOP)** exercises that I developed during my studies at **Senac**, focused on practicing the core concepts of object-oriented design.

## Purpose

These exercises were created to practice:

- creating classes and objects
- attributes and methods
- encapsulation
- code organization with separated responsibilities
- modeling real-world problems using OOP

## Repository exercises

- `01SistemaBancario.py` - Basic bank account operations simulation.
- `02ControleDeProdutos.py` - Simple product control and related information.
- `03SistemaDeFncionarios.py` - Employee modeling and basic characteristics.
- `04SistemaDeVeiculos.py` - Class structure for vehicles.
- `05Zoologico.py` - Animal representation and behavior in a zoo context.
- `06FormasGeometricas.py` - Geometric shape modeling and calculations.
- `07SistemaDePagamento.py` - Basic payment rules simulation.
- `08SistemaDeTransporte.py` - Simple transportation system modeling.

## Context

These projects were built for learning purposes to strengthen my OOP foundation, logical thinking, and code structuring skills in Python.
